# WhatsApp Confession Bot

Bot WhatsApp untuk mengirim confession secara anonim dengan sistem moderasi admin.

## Fitur
- ✅ Kirim confession anonim
- ✅ Sistem moderasi admin
- ✅ Approve/reject confession
- ✅ Post otomatis ke channel
- ✅ Multi-admin support
- ✅ Statistik lengkap
- ✅ Web dashboard (basic)

## Prasyarat
- Node.js 16.x atau lebih tinggi
- Akun WhatsApp (nomor HP)
- Akses ke terminal/command line
